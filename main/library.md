### Базовые ссылки <a name="base"></a>

1. [База вопросов на русском языке](https://db.chgk.info/). Туда можно добавлять пакеты и самостоятельно: в ручном режиме ([инструкция](https://db.chgk.info/add-questions)) или с помощью программы [chgksuite](#chgksuite).

2. Сайт [«Есть вопросы?»](https://gotquestions.online/) – зеркало Базы с более гибким поиском и [другими преимуществами](https://t.me/chgknews/495). Кроме вопросов есть [телеграм-канал с новостями](https://t.me/gotquestions_online), [чат поддержки](https://t.me/+kpfffc26n200MmVi) и [инструкция для загрузки пакетов](https://docs.google.com/document/d/1Rhq5RC_rzDZtINegFIqFkBL7G9yT6A2dk1lMm3B5Zbw/edit#heading=h.tvnjdhqezh12).

3. [База вопросов на украинском языке](https://www.bazaua.org.ua/) (очень давно не обновлялась).

4. Телеграм-чат для поиска тестеров. Вступить в чат можно по [этой ссылке](https://t.me/+BGSxZpfx9sIzODky). Обратите внимание: участникам нужно будет пройти верификацию у модераторов.

### Программы <a name="programs"></a>

1. <a name="chgksuite"></a>[chgksuite](https://chgksuite.pecheny.me/) — это система для работы с пакетами ЧГК. Она позволяет единообразно оформить вопросы, а также экспортировать их в разные форматы, в том числе с использованием спойлеров для ответов и комментариев. 
- [чат поддержки](https://t.me/chgksuite), 
- [видеоинструкция](https://youtu.be/dUQJCNTUYJk),
- [код на gitlab](https://gitlab.com/peczony/chgksuite).
2. Сергей Литвинов. [Trello для ЧГК. Часть 1: удобный способ хранить ваши вопросы и работать с ними](https://chgk-rostov.ru/articles/trello-for-chgk-intro)
3. Сергей Литвинов. [Trello для ЧГК. Часть 2: командная работа над пакетом вопросов](https://chgk-rostov.ru/articles/trello-for-chgk-team)
